# VBACode
一些VBA代码片段和小工具

## 内容
1. 文章语料分句处理程序：将爬虫采集的网络文章按照一定的规则智能分句。
2. 筛选未回复邮件：筛选outlook中还没有人回复你的邮件。
3. 加密货币新闻网址爬虫：爬取加密货币相关新闻的URL
4. 图片下载合并程序：将爬虫爬取的图片URL合并下载
5. 人工识别验证码简易查询工具：人工识别验证码，相当于自动填写表单工具
